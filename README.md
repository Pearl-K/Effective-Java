2. 해당 주차 정해진 분량의 책을 읽고 정리한다. (분량은 약 40page 내외)
    - Item 별로 구분해서 준비한다.
        - 이전에 몰랐던 것 중에 새로 알게된 부분
        - 본인이 중요하다고 생각한 부분 (정리할 필요가 있거나 강조할 만한 부분)
        - 관련 내용 중에서 코드에 적용해본 경험이 있거나 새로 적용해본 부분 (실습 코드)
3. 정리 플랫폼 자유
    - Github
    - Notion
    - 기타 기술 블로그나 코드, 내용 공유할 수 있는 플랫폼 가능
